# Movie Website

This project is a React-based web application focused on providing information about movies and news. The project includes a variety of features to enhance user experience.

## Features
* **Home Page** : Displays an overview of the latest movies and news.

## Technologies Used
* React
* React Router
* Tailwind CSS
* Icons from React Icons library

## Additional Details
* **Responsive Design** :  Adapts to different screen sizes, ensuring a user-friendly experience on mobile devices and tablets.
* **Dark/Light Mode** :Provides an option to toggle between dark and light themes, improving the user experience in various lighting conditions.
